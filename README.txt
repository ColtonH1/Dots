Features:
8 different colored circles, game can be played with anywhere from 2 to 8 colors
User has choice of how many rows and columns are in the grid (with a maxium of 8, so the grid won't go off the screen)
Particle Effect changes when player makes a loop
Player can make loops which will award all of the circles of the same color currently on the board
Countdown leading to a gameover state
Restart button to replay the game
Quit button---Note, there was no where to 'quit' to, so currently text just shows up to confirm the user quit
Score is kept and shown
Circles fall in zig-zag
Player can move backwards, undoing the line they just drew

and many, many more
